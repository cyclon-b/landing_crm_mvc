## Description

You can deploy it just with Node or using Docker as well all configuration is provided

## Installation

```bash
$ npm i
$ cd public_src && npm i
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Building frontend

```bash
# development
$ cd public_src && npm run build

# watch mode
$ cd public_src && npm run watch
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
